# Manual Install

# Wine
Wine is used to emulate a Windows environment to run the ShadowWorlds server.

Install Wine 32 to CentOS 7:
```
yum -y install https://harbottle.gitlab.io/wine32/7/i386/wine32-release.rpm
yum -y install wine.i686
```
